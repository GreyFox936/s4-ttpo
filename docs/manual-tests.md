# Ручные тест-кейсы F-Bank

## TC-001 — Отображение счетов при запуске сервиса

**Тест-сьют:** Smoke  
**Приоритет:** High  
**Предусловия:** сервис запущен командой `python3 -m http.server 8000`.

| № | Action | Expected Result | Test Result |
|---|---|---|---|
| 1 | Открыть `http://localhost:8000/?balance=30000&reserved=20001` | Открывается главная страница F-Bank | Passed |
| 2 | Проверить заголовок страницы | Отображается `F-Bank` | Passed |
| 3 | Проверить список счетов | Отображаются `Рубли`, `Доллары`, `Евро` | Passed |
| 4 | Проверить рублёвый счёт | Отображается баланс `30'000 ₽` и резерв `20'001 ₽` | Passed |

**Постусловия:** не требуются.
